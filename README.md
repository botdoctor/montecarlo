# montecarlo
A Monte Carlo simulation model in Python to forecast music performance metrics for a client, enabling data-driven decision-making on song releases. The model runs thousands of randomized scenarios to estimate probable outcomes across key variables, helping the business quantify risk and identify high-potential investments.


How to run
Clone first clone the repo
https://github.com/botdoctor/montecarlo
It is best practice to create a virtual environment for your dependencies.
run python -m venv venv
to activate the venv run 
.\venv\Scripts\activate on windows or
./venv/Scripts/activate on linux
then run
